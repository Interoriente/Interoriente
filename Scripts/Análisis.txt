Objetivos: 

1. Obtener información de la Api de Personas y almacenarla en la bd 
de manera automática. Siéndo "automática" que el servidor ejecute el script 
cada cierto tiempo. 
2. Obtener información (Título, precio, imagenes y descripcion) 
de los links almacenados en tblLinks y luego crear registro en 
tblPublicaciones teniendo en cuenta la información extraída asignando 
a un usuario al azar de la tabla tblUsuarios (En este caso su docId) la publicación
3. Simular una compra de una publicación. 

Para esto se debe almacenar el id de una publicación y el id de un usuario en la tabla tblFactura
